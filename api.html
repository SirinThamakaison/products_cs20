Google Search Results (Zenserp API)

Enter a search term and click Get Results to see what Google returns.

 

About this API

API: Zenserp Google Search API. It lets you get Google search results in JSON so you can use them in a project.

Why I picked it: I wanted to try an API that takes a parameter (the search query) and returns real data. Zenserp has a free plan and the docs were easy to follow.

Where I found it: https://app.zenserp.com/ (dashboard and documentation).

Parameters I used: In the query string I use q= for the search term (e.g. ?q=javascript%20tutorial) and apikey= for my API key. So the options/parameters are: (1) q = the search words, (2) apikey = my API key. Both go in the URL.

Example use: A small tool that checks what ranks on Google for a keyword, or a page that shows "what people are searching" for a topic. You could also use it to compare how different search terms show up in results.
